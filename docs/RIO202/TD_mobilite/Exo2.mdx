# Sélection et resélection 3G FDD

## Question 1

Il y a deux critères de selection:

-   1 critère de puissance (comme en 2G) - "signal assez fort"
-   1 critère de qualité - "signal assez bon"

Pour chaque cellules il faut calculer les 2 critères

## Question 2

SrXLevel = QRxLevMeas - QRxLevMin - PCompensation

|           | QRXLEVMEAS | PCompensation |      SrXLevel       | Squal = QqualMeas - QqualMin |
| :-------: | :--------: | :-----------: | :-----------------: | :--------------------------: |
| Cellule A |   -90dBm   |      0dB      |  -90-(-90)-0 = 0dB  |       -13-(-10) = -3dB       |
| Cellule B |   -85dBm   |  33-30 = 3dB  | -85-(-100)-3 = 12dB |        -7-(-12) = 5dB        |
| Cellule C |   -80dBm   |      0dB      | -80-(-90)-0 = 10dB  |       -10-(-10) = 0dB        |

Les cellules convenables sont SrxLvel > 0 et Squal > 0
