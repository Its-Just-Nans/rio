# RIO201 - Projet

Rapport individuel pour le 2 Décembre 2023

En utilisant le système de capteur / actionneur de FIT/IoT-LAB, créez votre banc d'essai simple de l'IoT

- Serveurs et clients HTTP/CoAP

- Informations de détection et automatisation
  - Capteurs : Température / Luminosité / Gyroscope, etc.
  - Actionneurs : écran d'impression / LED /boutons (clics)

- Imaginez une application IoT simple et définissez ses exigences


#### Vous n'avez pas besoin de réaliser tous les aspects d'une application IoT

- Fournir 3 fonctions d'automatisation de cette application IoT

    - Exemple) collecte d'informations sur la lumière et automatisation d'un signal pour allumer la lumière lorsqu'il devient sombre
    - Exemple) envoi d'une alarme si le capteur d'accélération change de valeur

- Cependant, Pour une fonction, utilisez le client CoAP sur votre SSH frontal

- Pour au moins une autre fonction, utilisez le client CoAP sur un nœud de capteur
