## TP4 - Complete virtualized network with open5Gs and srsRan_4G

Cette deuxième partie du TP4 a pour objectif de mettre en place un réseau 4G complet avec un core network open5Gs et un radio access network srsRan_4G avec des containers.

Le compte rendu, le code et l'explication de cette partie est sur ce repository: [https://github.com/Its-Just-Nans/dockerizing-4g](https://github.com/Its-Just-Nans/dockerizing-4g)
