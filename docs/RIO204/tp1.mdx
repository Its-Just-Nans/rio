# TP1 Vigie 2G

L'objectif de ce TP est d'étudier les principales procédures mises en œuvre sur une interface radio GSM. Pour les plus courageux d’entre vous, vous pourrez vous reporter avec profit aux recommandations 04.08 et 04.60 pour le GSM sur www.3gpp.org .
