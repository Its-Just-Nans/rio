# TP2 Vigie 3G


## Question de cours

<details>
<summary>1. Qu'est-ce qu'une connexion RRC ?</summary>

Un ensemble de SRB permettant à un utilisateur d'échanger de la signalisation avec le réseau

</details>

<details>
<summary>2. Entre quels équipements est-elle établie ? (2 choix)</summary>

- UE
- RNC

</details>

## Appel sortant

<details>
<summary>3. Quelle information d'identité se trouve dans le message RRC Connection Request ?</summary>

TMSI + LAI

</details>

<details>
<summary>4. Quelle est sa valeur ?</summary>

TMSI = 0010 0110 1110 1110 0111 1001 1110 0110
LAI = MCC + MNC + LAC = 208 + 20 + 0100 1110 0011 1111

</details>

<details>
<summary>37. Les Master Information Blocks :</summary>


- Sont décodés tout le temps
- Ne comportent pas d'informations systèmes
- Comportent la liste des SIB activés
- Comportent la liste des SIB modifiés

</details>

<details>
<summary>38. Les System Information Blocks (SIB) :</summary>

- Sont décodés en cas de changement
- Peuvent être découpés en plusieurs messages

</details>
