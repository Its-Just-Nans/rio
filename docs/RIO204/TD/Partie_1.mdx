# Training QCM

## 1 - Choisir la réponse correcte

- a. La 4G utilise un mécanisme d'authentification mutuelle entre le terminal et le réseau
- b. En 4G, seul le terminal s'authentifie auprès du réseau
- c. En 4G, seul le réseau s'authentifie auprès du terminal
- d. En 4G il n'y a pas de mécanismes d'authentification entre le terminal et le réseau


<details>
<summary>Réponse</summary>

- a. La 4G utilise un mécanisme d'authentification mutuelle entre le terminal et le réseau

---

- le réseau envoie un `rand()`
- le réseau envoie le `rand()`
- avec le `rand()` le téléphone va générer une valeur `authentification number`
- le réseau envoie son `authentification number`
- si la valeur est la même, le mobile à identifié le réseau.

---

- avec le `rand()` le téléphone calcule le `RES`
- le réseau calcule le `RES`
- le téléphone envoie le `RES` au réseau
- si c'est le même c'est authentifié

</details>
