# RIO204 - Réseaux mobiles

RIO204 contains 3 TPs

- TP 2G on Vigie
- TP 3G on Vigie
- TP 4G with Amarisoft Software


## Notation

Note finale de l'UE = Note du CC/18 + Moyenne TP /2

- Le CC est divisé en 3 parties 2G, 3G, et 4G notée chacune sur 6 points
- Les 3 TP Vigie 2G, Vigie 3G et Amarisoft 4G comptent pour le même poids dans la moyenne des TP
